Présentation du projet WEB

Contexte du projet

Suite à la fusion des entités Galaxy et Swiss-Bourdin en 2009, le groupe nouveau GSB est devenu leader du marché pharmaceutique. Afin de faciliter le stockage, GSB a besion d'un outil de gestions de stockage en local et sécurisé. L'objectif de l'outil est de centraliser tout le stockage afin d'avoir des informations importantes sur le suivi.

Pour commencer une simple page d’accueil pour vous connecter

Après la connexion vous allez pouvoir rechercher, ajouter, voir l'historique (sur le stockage), vous connecter à l'application web, modifier votre compte et voir les utilisateurs (pour les administrateurs).

Pour l'historique vous avez le total de toutes les opérations effectuées

Pour ajouter un médicament il vous suffit d'aller dans l'onglet "Nouveau".
la particularité de cette page c'est qu'elle réalise un appel web service sur une page en php.

Pour vendre des médicaments il vous suffit de cliquer sur un médicament afin d'ouvrir la fenêtre de gestion des produits. Vous pouvez vendre des produits ou le modifier.

Mes Applications

Visual Studio

L'outil que j'ai utilisé pour mes projets web est visual studio code. Visual Studio Code est un éditeur de code source développé par Microsoft pour Windows , Linux et macOS . Il inclut la prise en charge du débogage , du contrôle Git intégré et de GitHub , de la coloration syntaxique , de l'achèvement intelligent du code , des extraits de code et de la refactorisation du code . Il est hautement personnalisable, permettant aux utilisateurs de modifier le thème , les raccourcis clavier , les préférences et d'installer les extensions qui ajoutent des fonctionnalités supplémentaires. Visual studio code

Wampserver

Pour créer mon environement de test, j'ai utilisé Wampserver. WAMP Signifie «Windows, Apache, MySQL et PHP». WAMP est une variante de LAMP pour les systèmes Windows et est souvent installé sous forme de bundle logiciel (Apache, MySQL et PHP). Il est souvent utilisé pour le développement Web et les tests internes, mais peut également être utilisé pour servir des sites Web en direct. Wampserver
