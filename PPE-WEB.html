Présentation du projet WEB

Contexte du projet

Suite à la fusion des entités Galaxy et Swiss-Bourdin en 2009, le groupe nouveau GSB est devenu leader du marché pharmaceutique. Afin de faciliter les test de medicaments sur des patients, GSB a besion d'un outil de gestions des patients en ligne et sécurisé. L'objectif de l'outil est de centraliser tous les patients afin d'avoir des informations important sur les résultats des médicaments qu'ils aurons testé.

Pour commencer une simple page d’accueil avec deux liens, vous pouvez choisir de vous connecter ou de vous enregistrer

Pour la page inscription vous allez pouvoir vous inscrire. tous les champs sont obligatoires afin de terminer l'inscription. votre compte est très important car vous allez pouvoir vous conneter sur les deux applications avec le compte.

Pour vous connecter rien de plus simple il vous sufit de vous connecter avec le compte que vous avez créé.

Suite à votre connexion vous allez avoir accès à la page suivante.


Vous allez pouvoir voir les patients sur la droite. Vous pouvez seulement voir vos patients.
Pour regarder vos patients il vous suffit de cliquer sur le patient à droite, et sont IBM sera calculer automatiquement.

Sur la dernière page vous allez pouvoir modifier et ajouter des patients

Mes Applications

Visual Studio Code

L'outils que j'ai utilisé pour mes projets web est visual studio code. Visual Studio Code est un éditeur de code source développé par Microsoft pour Windows , Linux et macOS . Il inclut la prise en charge du débogage , du contrôle Git intégré et de GitHub , de la coloration syntaxique , de l'achèvement intelligent du code , des extraits de code et de la refactorisation du code . Il est hautement personnalisable, permettant aux utilisateurs de modifier le thème , les raccourcis clavier , les préférences et d'installer les extensionsqui ajoutent des fonctionnalités supplémentaires. Visual studio code

Wampserver

Pour créer mon environement de test, j'ai utilisé Wampserver. WAMP Signifie «Windows, Apache, MySQL et PHP». WAMP est une variante de LAMP pour les systèmes Windows et est souvent installé sous forme de bundle logiciel (Apache, MySQL et PHP). Il est souvent utilisé pour le développement Web et les tests internes, mais peut également être utilisé pour servir des sites Web en direct. Wampserver
